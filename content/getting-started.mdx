# Getting Started

## Installation

```bash
npm install @boooply/ai-interviews-sdk
```

## Authentication

All API requests require an API key. You can get your API key from the [Boooply Integrations page](https://boooply.com/integrations) or by contacting support.

```bash
# Store in your .env file
BOOOPLY_API_KEY=bply_org_xxxxxxxxxxxxx
BOOOPLY_BASE_URL=https://api.boooply.com
```

> **Security:** Never expose your API key in client-side code. Always call the Boooply API from your backend.

## Initialize the client

```javascript
const { BoooplyClient } = require('@boooply/ai-interviews-sdk');

const client = new BoooplyClient({
  apiKey: process.env.BOOOPLY_API_KEY,
  baseUrl: process.env.BOOOPLY_BASE_URL,
  timeout: 30000, // optional, default 30s
});
```

## Create your first interview

```javascript
const { meeting } = await client.interviews.create({
  type: 'ai',
  jobRole: 'Frontend Engineer',
  candidate: {
    name: 'Jane Doe',
    email: 'jane@example.com',
  },
  skills: ['React', 'TypeScript', 'CSS'],
  durationMinutes: 20,
});

console.log('Interview created:', meeting.meetingCode);

// Build the candidate's tokenized join URL (Boooply also emails this automatically)
const candidate = meeting.participants.find(p => p.role === 'CANDIDATE');
const joinUrl = `${meeting.url}?token=${candidate.joinToken}`;
console.log('Candidate join URL:', joinUrl);
```

The candidate opens the URL in their browser, the camera comes on (required — can't be toggled off during the session), and the AI interviewer takes over — asking questions, evaluating responses, and generating a detailed assessment.

## Get results

After the interview completes, retrieve the evaluation. The analysis job runs asynchronously, so `evaluation` may be `null` until it finishes — subscribe to the `interview.evaluation` [webhook](/webhooks) to be notified instead of polling.

```javascript
const result = await client.interviews.getEvaluation(meeting.meetingCode);

if (result.evaluation) {
  console.log('Score:', result.evaluation.overallScore);          // 0-100
  console.log('Recommendation:', result.evaluation.recommendation); // PASS | FAIL | REVIEW
  console.log('Communication:', result.evaluation.communication);
  console.log('Technical:', result.evaluation.technical);
  console.log('Culture Fit:', result.evaluation.cultureFit);
  console.log('Problem Solving:', result.evaluation.problemSolving);
  console.log('Experience Match:', result.evaluation.experienceMatch);
}
```
